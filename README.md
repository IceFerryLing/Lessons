# HITsz 资料分享

## 项目简介
本仓库整理了哈尔滨工业大学（深圳）学生在学习过程中积累的课程资料、作业示例与复习资源，方便同学们查阅与互助。内容涵盖基础课、专业课以及考研复习素材，旨在为学习与备考提供一个集中、开放的参考入口。

## 仓库结构概览
- `计算思维与实践/`：课程作业与实验示例（如排序算法、递归、栈/队列综合实验、字符串处理等）。
- `大学物理/`：与大学物理课程相关的资料。
- `数学分析上/`、`数学分析下/`：习题、试卷、作业答案等。
- `线性代数/`：期中期末试卷与作业资料。
- `人工智能导论/`：AI 相关课程资料。
- `考研复习资料/`：考研相关的复习资料合集。
- `新思想概论（习概）/`：思想政治课程资料。

> 目录会随着资料更新而扩充，文件名基本按课程章节或实验编号命名，便于快速定位。

## 使用建议
- 仅作为学习与复习参考，请结合课堂讲授、教材与官方资料理解和消化内容。
- 若需要下载整库，可直接克隆或打包下载；也可按需获取单个子目录。
- 保持原有目录与命名，不随意改动文件结构，便于他人定位与协作。

## 贡献方式
- 欢迎补充更多课程资料、习题解析或学习笔记，可直接按现有目录增加文件。
- 建议在文件名中注明课程、章节或日期，确保可读性与可检索性。
- 上传前请确认资料来源合规，避免侵犯他人版权或泄露个人隐私。
